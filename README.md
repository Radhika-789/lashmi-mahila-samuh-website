#  Lashmi Mahila Samuh Website

A web-based platform designed to support and showcase the work of **Lashmi Mahila Samuh**, empowering women through digital presence, product visibility, and community engagement.

---

##  Project Overview

The **Lashmi Mahila Samuh Website** aims to:

- Promote women-led self-help group initiatives  
- Showcase handmade and local products  
- Provide information about the organization  
- Enable users to connect with the group  

This project focuses on **social impact and digital accessibility**.

---

## Features

-  Homepage – Overview of the organization  
-  About Section – Information about the Mahila Samuh  
-  Products Section – Display of handmade products  
- Contact Page – Communication details  
-  Responsive Design – Mobile-friendly interface  

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Styling** CSS Flexbox / Grid 
- **Version Control**  Git, GitHub
